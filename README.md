# minesweeper



## settings
    works for columns and rows

## instructions

    left click ->  
    open cell
   
    right click ->
    flag cell as mine
  
    right click again ->
    flag cell as unknown
  
    right click again ->
    remove flag

## customize tiles
    insert pictures in main/resources/original_images
    run main/utils/imagesDrivers/ResizeImages
 
   supports .jpg and .png
